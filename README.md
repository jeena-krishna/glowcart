# GlowCart

A full-stack clean beauty e-commerce platform built with Spring Boot and React.

## Tech Stack

**Backend:** Java 17, Spring Boot 3.4, Spring Security, Spring Data JPA, PostgreSQL
**Frontend:** React 18, Vite, Tailwind CSS (coming soon)
**Payments:** Stripe (test mode)
**Deployment:** Docker, Docker Compose

## Project Status

🚧 Under active development

## Features (Planned)

- [ ] User authentication with JWT
- [ ] Product catalog with real beauty product data
- [ ] Ingredient transparency and clean beauty scoring
- [ ] Shopping cart with stock validation
- [ ] Secure checkout with Stripe
- [ ] Admin dashboard for product management
- [ ] Skin-type based product recommendations
